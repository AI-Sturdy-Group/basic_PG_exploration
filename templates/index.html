Reinforcement Learning - Base Policy gradient in simplistic environment

Agent

We are aiming to create and train a stochastic agent. This means that the agent doesn't gives a single action to take in a state, but instead it outputs a probability distribution of taking different actions. Then we pick a single action from that distribution, and take a step in the environment.

The environment is continuous so we don't have a finite number of actions to chose from, we have a range of possible action values (e.g. [-4, 4]), and we can chose any number from it as our action in each step (e.g. 1,541).

We will design the agent to output a Gaussian Distribution and we will pick action from there. The distribution is completely defined by it's mean (μ) and variance (σ), so we need our agent to be able to output this two values.
